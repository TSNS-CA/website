# Turkish Society of Nova Scotia Frontend

Frontend for the brand new Turkish Society of Nova Scotia website.
 
This repository contains a minimal React + Vite starter scaffold (JSX) optimized for desktop and mobile.

Run locally:

```bash
npm install
npm run dev
```
